# MeModify

日韓翻訳テキスト(SRT/TXT)から韓国語の行だけを削除し、日本語とタイムスタンプのみを残す、高機能なテキスト加工ツールです。

## 主な機能

- **韓国語行の自動削除**: タイムスタンプの直後に来る韓国語の台詞行を自動的に識別して削除します。複数行の韓国語が続く場合も正確に処理します。
- **データの保持**: タイムスタンプ、日本語翻訳、インデックス番号、改行などはオリジナルのまま正確に維持します。
- **インテリジェントな例外処理**: 「의역」(意訳)、「직역」(直訳)、「순화」(純化)などのキーワードが含まれる行は、重要な補足情報として自動的にハイライト表示され、削除対象から除外されます。
- **リアルタイム編集 & 復元**: 加工結果を画面上で直接修正できるだけでなく、自動削除された行をワンクリックで復元することも可能です。
- **フル画面モード**: 広い画面領域を活用して、大量の字幕データを効率的にチェック・編集できます。
- **プライバシー保護**: すべての処理はブラウザ内で完結し、サーバーへデータが送信されることはありません。
- **デスクトップ対応**: インストール不要の単一実行ファイル(.exe)として、ローカル環境ですぐに使用できます。

## 使用方法

1. `MeModify.exe` を実行します。
2. 加工する `.txt` または `.srt` ファイルを画面にドラッグ＆ドロップするか、クリックしてアップロードします。
3. **削除行（赤）**と**保持行（通常）**、**重要行（黄）**を確認します。
4. 必要に応じて、行のテキストを直接編集したり、復元ボタンで削除を取り消したりします。
5. 「ダウンロード」ボタンを押して結果を保存（ファイル名に `_fixed` が付与されます）するか、「クリップボード」でコピーします。

## 技術スタック

- **Frontend**: React, Vite
- **Desktop**: Electron
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (Glassmorphism design)

## ビルドおよび実行

```bash
# 依存関係のインストール
npm install

# 開発モードの実行 (Vite)
npm run dev

# デスクトップアプリの開発モード実行
npm run electron:dev

# デスクトップアプリのビルド (Portable .exe)
npm run electron:build
```

# MeModify (한국어)

한일 번역 텍스트(SRT/TXT)에서 한국어 행만 제거하고 일본어와 타임스탬프만 남겨주는 고성능 텍스트 가공 도구입니다.

## 주요 기능

- **한국어 행 자동 제거**: 타임스탬프 바로 뒤에 오는 한국어 대사 행을 자동으로 식별하여 제거합니다. 여러 행의 한국어가 이어지는 경우에도 정확하게 처리합니다.
- **데이터 보존**: 타임스탬프, 일본어 번역, 인덱스 번호, 줄바꿈 등은 원본 그대로 정확하게 유지합니다.
- **지능형 예외 처리**: 「의역」, 「직역」, 「순화」 등의 키워드가 포함된 행은 중요한 보충 정보로 자동 하이라이트 표시되며 삭제 대상에서 제외됩니다.
- **실시간 편집 및 복원**: 가공 결과를 화면에서 직접 수정할 수 있을 뿐만 아니라, 자동 삭제된 행을 클릭 한 번으로 복원할 수도 있습니다.
- **전체 화면 모드**: 넓은 화면 영역을 활용하여 대량의 자막 데이터를 효율적으로 확인하고 편집할 수 있습니다.
- **프라이버시 보호**: 모든 처리는 브라우저 내에서 완결되며 서버로 데이터를 전송하지 않습니다.
- **데스크톱 지원**: 설치가 필요 없는 단일 실행 파일(.exe)로 로컬 환경에서 바로 사용할 수 있습니다.

## 사용 방법

1. `MeModify.exe`를 실행합니다.
2. 가공할 `.txt` 또는 `.srt` 파일을 화면으로 드래그 앤 드롭하거나 클릭하여 업로드합니다.
3. **삭제된 행(빨간색)**, **남은 행(일반)**, **중요 행(노란색)**을 확인합니다.
4. 필요시 텍스트를 직접 편집하거나 복원 버튼으로 삭제를 취소합니다.
5. `다운로드` 버튼을 눌러 결과물을 저장(파일명에 `_fixed`가 추가됨)하거나 `클립보드` 버튼으로 복사합니다.

## 기술 스택

- **Frontend**: React, Vite
- **Desktop**: Electron
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (Glassmorphism design)

## 빌드 및 실행

```bash
# 의존성 설치
npm install

# 개발 모드 실행 (Vite)
npm run dev

# 데스크톱 앱 개발 모드 실행
npm run electron:dev

# 데스크톱 앱 빌드 (Portable .exe)
npm run electron:build
```
