# MeModify

日韓翻訳テキスト(SRT/TXT)から韓国語の行だけを削除し、日本語とタイムスタンプのみを残すテキスト加工ツールです。

## 主な機能

- **韓国語行の自動削除**: タイムスタンプの直後に来る韓国語の台詞行を自動的に識別して削除します。
- **データの保持**: タイムスタンプ、日本語翻訳、インデックス番号、改行などはオリジナルのまま維持します。
- **例外処理**: 「의역：」(意訳) または 「직역：」(直訳) で始まる行は、韓国語であっても削除せずに保存します。
- **リアルタイム編集**: 加工された結果を画面上で直接修正でき、修正した状態でそのままダウンロードやコピーが可能です。
- **プライバシー保護**: すべての処理はユーザーのブラウザ内でローカルに実行され、サーバーにデータが送信されることはありません。
- **デスクトップアプリ**: インストール不要の単一実行ファイル(.exe)として、ローカル環境で手軽に使用できます。

## 使用方法

1. `MeModify.exe` を実行します。
2. 加工する `.txt` または `.srt` ファイルを画面にドラッグ＆ドロップするか、クリックしてアップロードします。
3. 削除された行（赤色の打ち消し線）と残った行を確認します。
4. 必要に応じてテキストを直接クリックして修正します。
5. 「ダウンロード」ボタンを押して結果を保存するか、「クリップボード」ボタンでコピーします。

## 技術スタック

- **Frontend**: React, Vite
- **Desktop**: Electron
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (Glassmorphism design)

## ビルドおよび実行

```bash
# 依存関係のインストール
npm install

# 開発モードの実行 (Vite)
npm run dev

# デスクトップアプリの開発モード実行
npm run electron:dev

# デスクトップアプリのビルド (Portable .exe)
npm run electron:build
```
